Gofem

Go Finite Element Method

Download this project as a .zip file Download this project as a tar.gz file

Welcome to Gofem, an implementation of the Finite Element Method in Go (golang).

This project includes a number of routines to develop the FEM for solid and porous media. Our targeted simulations include the static and dynamics of structures, multi-phase fluid flow through porous media, and the behaviour of porous media considering the coupling between liquid, gas and solids. Some sub-packages are created to make the FEM implementation easier; these include:

  • ana analytical solutions
  • shp shape structures and quadrature points
  • inp input data structures. simulation, materials, meshes
  • msolid models for solids
  • mconduct models for liquid/gas conductivity in porous media
  • mreten models for liquid retention in porous media
  • mporous models for porous media
  • fem finite element method
  • out results analyses and plotting

Installation

Gofem is developed in/for Debian systems at the moment. It depends on Gosl that is available here. See also http://cpmech.github.io/gosl

After installing Go (golang) and Gosl, use the type the following commands

mkdir -p $GOPATH/src/github.com/cpmech
cd $GOPATH/src/github.com/cpmech
git clone https://github.com/cpmech/gofem.git
cd gofem
./all.bash
